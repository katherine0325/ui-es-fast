5分钟开始一个应用

欢迎使用 ESFast！本指南将帮助你快速上手并开始构建你的桌面应用。

安装

使用 npm 或 yarn 安装 ESFast：

npm install esfast

创建新项目

使用 ESFast CLI 创建一个新项目：

npx esfast create my-app

启动开发服务器

进入项目目录并启动开发服务器：

cd my-app
npm run dev

开始开发

现在你可以开始开发你的应用了。ESFast 提供了以下功能：

  • 内置 Ant Design 组件库
  • Tailwind CSS 支持
  • 路由系统
  • 数据存储解决方案
  • Node.js 原生方法调用
  • 图片管理工具
