# 5分钟开始一个桌面应用

## 介绍

ESFast 是一个桌面应用基础开发框架，它集成了 Vue3，Antd，TailwindCss，Typescript，Indexdb存储数据，Node原生能力调用，图片存储最佳实践，让你能够快速开始构建高质量的桌面应用。


## 安装依赖

```bash
# npm
npm install
# 或
# yarn
yarn
```

## 启动开发模式

```bash
# npm
npm run dev
# 或
# yarn
yarn dev
```


这将启动开发服务器，并打开一个窗口，显示你的应用。当你修改代码时，应用会自动重新加载。
![image](/docs/images/start_01.jpg)
