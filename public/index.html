快速构建桌面应用

ESFast 是一个基于 Electron 的桌面应用开发框架，集成了 Vue3、Ant Design、Tailwind CSS 和 TypeScript，提供开箱即用的开发体验。

开始使用 GitHub

核心特性

开箱即用

内置 Vue3、Ant Design、Tailwind CSS 和 TypeScript，无需繁琐配置，立即开始开发。

数据存储

内置 IndexedDB 存储方案，提供完整的增删改查方法，让数据操作变得简单。

文件操作

便捷调用 Node.js 文件系统方法，轻松处理本地文件。

图片管理

优化的图片存储方案，支持简短地址映射展示。

打包部署

已配置好打包程序，支持一键打包，适用于 Mac 和 Windows 平台。

开发体验

热更新、TypeScript 支持、代码提示，提供流畅的开发体验。
